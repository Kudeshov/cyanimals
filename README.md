# Животные Кипра — статический сайт

## Что это такое
Это простой сайт для документирования случаев насилия над животными на Кипре. Сайт показывает инциденты на карте, позволяет фильтровать их и отправлять новые сообщения через форму.

## Структура файла `incidents.json`
Каждый инцидент хранится в виде объекта JSON с полями:
- `id` — уникальный идентификатор, например `nicosia-2026-04-28-001`
- `city` — город, например `Nicosia`
- `district` — район или местность, например `Pera Chorio Nisou`
- `title` — заголовки на трех языках: `title.ru`, `title.en`, `title.el`
- `date` — дата в формате `YYYY-MM-DD`
- `lat`, `lng` — координаты широты и долготы
- `type` — тип инцидента: `shooting`, `poisoning`, `other`
- `status` — статус: `reported`, `pending`, `confirmed`, `injured`, `dead`
- `description` — описание на трех языках: `description.ru`, `description.en`, `description.el`
- `images` — массив путей к фотографиям, например `["incidents/nicosia-2026-04-28-001/1.jpg", "incidents/nicosia-2026-04-28-001/2.jpg"]`

## Как добавить инцидент через VS Code AI помощника
1. Установите расширение GitHub Copilot в VS Code.
2. Откройте файл `incidents.json` в редакторе.
3. Напишите в чате Copilot запрос, например: "Добавь новый инцидент: собака по кличке Шарик, найдена в Никосии, ранена пулей, дата 2026-04-29, координаты 35.123, 33.456, 2 фото."
4. Copilot сгенерирует JSON-объект и заменит или добавит запись прямо в файл `incidents.json`.
5. Если нужны переводы, попросите Copilot: "Переведи заголовок и описание на английский и греческий."
6. Попросите помощника создать папку для фото: `incidents/ваш-id/`.
7. Поместите фото в формате `jpg` в папку: `1.jpg`, `2.jpg` и т.д.
8. Сохраните изменения.

## Как проверить сайт локально с Live Server
1. Установите расширение `Live Server` в VS Code.
2. Откройте папку проекта в VS Code.
3. Нажмите правой кнопкой на `index.html` и выберите `Open with Live Server`.
4. Сайт откроется в браузере, и изменения будут показываться сразу после сохранения файлов.

## Как работать с GitHub
### Установка Git
Скачайте и установите Git с официального сайта: https://git-scm.com/downloads. Это приложение для работы с репозиториями.

### Как взять исходники
1. Перейдите на страницу репозитория: https://github.com/Kudeshov/cyanimals
2. Нажмите кнопку "Code" и скопируйте URL.
3. В терминале выполните: `git clone <URL>`
4. Откройте папку в VS Code.

### Как коммитить изменения
1. Внесите изменения в файлы (например, добавьте новый инцидент).
2. В терминале: `git add .`
3. `git commit -m "Добавлен новый инцидент с собакой"`
4. `git push origin main` (или ваша ветка)

## Правило о личных данных
Не публикуйте:
- Полные домашние адреса
- Номера телефонов
- Имена частных лиц (без их согласия)

## Что входит в пакет
- `README.md` — документация и инструкция по работе с проектом
- `index.html` — главная страница сайта
- `incidents.json` — все инциденты хранятся здесь в формате JSON
- `assets/css/styles.css` — стили оформления
- `assets/js/app.js` — код для переключения языка, карт, отображения и отправки формы
- `incidents/<id-инцидента>/` — папки с фотографиями (опционально)
- `PROMPT_TEMPLATE.md` — шаблон для добавления нового инцидента с помощью ИИ
